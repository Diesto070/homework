# Проект «Виджет финансовых операций»

## Описание:

Проект «Виджет финансовых операций» это виджет, который показывает несколько последних успешных банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
```
https://github.com/Diesto070/homework.git
```
2. Установите зависимости:
```
pyproject.toml
```
## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Создайте новый проект и начните добавлять задачи.
